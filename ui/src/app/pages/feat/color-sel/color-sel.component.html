颜色选择器

基于 ng-zorro-antd NzColorPickerModule 实现，支持多种格式与尺寸配置。

基本用法
{{ color() }}
HEX 格式
{{ hexColor() }}
RGB 格式
{{ rgbColor() }}
HSB 格式
{{ hsbColor() }}
大尺寸
{{ largeColor() }}
默认尺寸
{{ defaultColor() }}
小尺寸
{{ smallColor() }}
禁用透明度
{{ noAlphaColor() }}
显示颜色文本
Hover 触发
{{ color() }}

注意：ng-zorro-antd 的颜色选择器是原生组件，相比 ngx-color-picker 功能更精简。 不支持灰度模式、吸管工具、自定义按钮样式等高级功能。如需这些功能，可以考虑其他第三方库或自定义实现。
